# 2D Top-Down Action Game

A fast-paced top-down action game inspired by **Hotline Miami**. Take on the role of a mysterious protagonist, battling through intense levels filled with enemies, bullets, and chaos. Precision, strategy, and lightning-fast reflexes are key to surviving.

**Created and maintained by Jad-d-Hamad**

---

## 🎮 Features

- **Top-Down Combat:** Navigate intricate levels from a top-down perspective.
- **Fast-Paced Action:** Quick kills, instant respawns, and non-stop intensity.
- **Stylized Graphics & Soundtrack:** Retro-inspired visuals and adrenaline-pumping music.
- **Multiple Levels & Challenges:** Each stage tests your skills and timing.

---

## 🛠 How to Run

1. Open the project in **GameMaker Studio**.
2. Build/run the game for your platform (Windows, HTML5, etc.).
3. Play and enjoy!
